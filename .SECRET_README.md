# 🤫 Secret Build Instructions

Este es tu documento secreto para compilar **llama-cpp-turboquant** localmente, optimizado para tu arquitectura x86_64 con aceleración CUDA, sin depender de los agentes.

## Requisitos Previos

Asegúrate de estar en el directorio raíz del repositorio:
```bash
cd /home/ignatus/GitHub/llama-cpp-turboquant
```

## Instrucciones de Compilación

Dado que el antiguo sistema de compilación `make` fue reemplazado, ahora dependemos de **CMake**. Los siguientes comandos configurarán y compilarán el proyecto entero aprovechando todos los núcleos de tu procesador.

### 1. Configurar el proyecto
Ejecuta esto para inicializar el directorio de compilación. Detectará automáticamente tu GPU y la arquitectura (CUDA 86-real, GNU Compiler):
```bash
cmake -B build .
```

### 2. Compilar
Este comando construirá el binario utilizando 8 hilos en paralelo para mayor velocidad:
```bash
cmake --build build --parallel 8
```

## ¿Cómo ejecuto los tests si algo falla?
Si modificas el código y quieres asegurarte de que TurboQuant y la gestión de memoria siguen funcionando al 100%, corre:
```bash
./build/bin/test-quantize-fns
```
